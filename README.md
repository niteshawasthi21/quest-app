# Quest App

A browser-based Vite + React quiz application that uses JSON question data.

## Features
- Registration and login using name, email, and password.
- 150 question exam loaded from local JSON.
- 60 minute timer for the full exam.
- Free question navigation.
- Flag questions for review.
- Manual submit before time ends.
- Auto-submit when timer reaches zero.
- Max 3 attempts per user.
- 80% pass rule.
- Detailed report page after exam.
- Context API state management.
- localStorage-based browser persistence.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
